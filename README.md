# 🏠 First Home Guide – Design Thinking Project

## 📌 Overview
This project was developed as part of a Design Thinking course with the goal of addressing a real-world problem:

> ❗ **Lack of information among young adults when buying their first home**

The platform is designed as an **interactive, user-centered web experience** that helps users understand, plan, and simulate the process of purchasing a house.

---

## 🎯 Problem Statement
Young adults (18–30) often face:

- Lack of financial knowledge  
- Confusion about credit systems (TAN, TAEG)  
- Uncertainty about affordability  
- Limited awareness of government support  
- Anxiety due to the complexity of the process  

---

## 📊 Key Insights (Based on User Research)
- 📉 **~70%** do not believe they can buy a house in the next 5 years  
- 💰 **~80%** identify financial issues as the main obstacle  
- ⚠️ **~83%** consider the process complex  
- ❓ ~50% are unaware of available support programs  
- 🧠 **~80%** say this impacts major life decisions  

---

## 👤 User Persona
A typical user is:

- A young adult with little to no experience in buying property  
- Feels overwhelmed and uncertain  
- Seeks simple, clear, and reliable information  
- Uses mobile devices frequently  
- Needs guidance before making financial decisions  

---

## 💡 Solution
The project proposes a **modern interactive website** that:

- Simplifies complex financial concepts  
- Provides real-time simulations  
- Visualizes research insights  
- Guides users step-by-step  
- Creates a calm and intuitive user experience  

---

## ⚙️ Features

### 🧮 Interactive Tools
- Mortgage Calculator  
- Affordability Estimator *(planned)*

### 📊 Data Insights
- Visual representation of research data  
- Simple explanations of key findings  

### 📚 Learning Section
- Clear explanations of:
  - TAEG  
  - TAN  
  - Mortgage process  

### ❓ FAQ Section
- Expandable answers  
- Based on real user concerns
